wordpress 3.3.2 帅美来袭，ready？

发表于 2012-04-25   |   分类于 wordpress   |  

相识wordpress是在3.3.1版本，什么？我几岁了？嗯，我不狡辩，新手上路，多多包涵。更多口水请对准《wordpress到底火了多久？我落后了多久？》，呵呵。前几天迎来了wordpress 3.3.2的帅美升级，马上体验吧。

wordpress3.3.2是对以往版本的安全升级：

  1. Plupload（1.5.4 版本） WordPress 用它来完成媒体文件的上传。
  2. SWFUpload WordPress 原先使用的库，用于上传媒体文件，一些插件可能仍在使用它。

  3. SWFObject — WordPress 原先用来嵌入 Flash 内容的库，一些插件和主题可能仍在使用它。


    修补点：

  4. 在 WordPress “站点网络”功能启用的环境下，站点管理员有可能在整个站点网络范围停用某个插件。

  5. URL 自动链接的算法可能允许跨站脚本攻击。
  6. 过滤 URL 的算法可能允许跨站脚本攻击。
    升级前注意备份整个网站，包括数据库。如果平时养成定时备份的好习惯，上天会在关键时刻眷顾我们的。

    感谢升级到最新版本！使用 WordPress 3.3.2 不仅可以让男生更帅，女生更美、改良人格，还可以大幅提升您的网上发布体验。诶，开玩笑的，其实她只能提升您的使用体验，但是光这一点就很不错了。

    开始了吗？已经结束了。wordpress的升级是如此简单。
    wordpress更有活力，大家更有动力！
