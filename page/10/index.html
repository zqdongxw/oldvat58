Fcitx 4.2.0版较老版有了很大的改进，内置了GOOGLE拼音。

删除ibus输入法：sudo apt-get remove ibus
安装FCITX： 
sudo apt-get install fcitx fcitx-pinyin fcitx-sunpinyin  fcitx-table

需要重启。

为Fcitx增加云拼音的支持
