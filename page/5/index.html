中国人为什么丧失了慢的能力？资源紧缺引发争夺，分配不平衡带来倾轧，速度带来烦躁，便利加重烦躁，时代的心态就是再也不愿意等。

不可不知的lollipop的4条安全特性

发表于 2014-12-04 | 分类于 Android |

现在有越来越多的朋友吃上了棒棒糖，可是关于lollipop的几个安全特性你了解吗？我来帮你一一解析。

1. 信任设备和信任面孔

在lollipop中安卓新增了“smart lock” 智能锁新特性，在设置安全里边找到smart lock ，其中包含信任的设备和信任的面孔。信任的设备依赖于蓝牙和NFC，当手机跟已经信任的设备，例如智能手表、车载系统连接时，手机会认为这是一个安全的环境，会保持解锁的状态。这就意味着当你驾车或者带着智能手表在办公室工作时，再也不用频繁地解锁了。如果断开了与信任设备之间的连接，手机会自动锁定。配合NFC标签会有意想不到的效果呢。

再来说一下信任的面孔。首先设置信任的面孔，设定成功，注意体会下这时手机与脸之间的距离及角度，有助于解锁时提高成功率。当你需要解锁的时候，面对摄像头几秒钟即可解锁（一定要注意距离，保证整个脸被摄像头捕捉到）。特别说明，人脸识别状态下，屏幕下方圆圈中会出现一个人像，识别成功后会出现一把解锁的锁。非识别状态，圆圈中是一把锁，任你什么角度都没用。如果失败，可以通过PIN码等方式来解锁。so cool , 亲测有效，即便是风头已过的i9300格调又重新拉高了有没有？抓紧向你的朋友秀一下吧。

 

2. 通知的隐私控制

我们知道，在lollipop的锁屏界面上会展现应用通知，比如wechat、G+ 、whatsapp 等应用推送的消息，这样一来更加方便了，但是隐私问题随之而来。幸好lollipop提供了完善细腻的隐私控制功能。在设置-提示音和通知 里，我们可以设定设备锁定时通知的状态，有三项可以选择：显示所有通知内容、隐藏敏感通知内容、完全不显示通知。

其中，隐藏敏感通知内容应该是最具有应用场景的基于应用级别的设定方式了。在此选项下，当有新的通知时，你会知道有了一封新邮件、一条聊天的回复，但是任何人看不到具体内容甚至是内容来源，提示内容已隐藏。

lollipop在应用通知里有了更为精细的针对每一个已安装应用的控制。也提供了三个选项：屏蔽（一律不显示来自此应用的通知）、优先（在列表顶部显示通知，并且在设备设置为“仅限优先打扰内容”时也显示）、敏感（来自此应用的敏感通知内容不会在设备锁定时显示）。这样一来，会满足你这样的应用场景：在锁屏界面，隐藏掉支付宝的敏感内容，同时可以便捷的看到不太敏感的QQ留言。

3. 屏幕固定（Screen Pinning）

屏幕固定功能可以把你的设备屏幕固定在一个app上，成为一个app的设备。其他人不能离开此固定的APP查看其它信息，也不能打开下拉菜单。

如何激活屏幕固定功能？在设置-安全-屏幕固定选项里开启此功能。然后可以在最近应用列表里点击第一个应用右下角的图钉按钮，即可锁定此应用。解锁的时候，需要同时触摸并按住“返回“和”概览“按钮。我发现解锁时几次未果，估计跟快捷键的设定有关系。这时候重启即可恢复。

4. 访客模式

Lollipop改善了支持多用户账户功能，为手机和平板设备提供了一种全新的Guest Mode访客模式。Guest Mode给每一位设备使用者都提供了安全绿色的一次性的操作空间，并且可以随时在guest account（访客账户）中删除用户数据。

8个必学软技能，让生活更美好

发表于 2014-11-27 | 分类于 V文摘 |

感谢我们曾经的学校和老师教给我们的东西，经历了启蒙、学习成长的过程。但是有些东西是学校里学不到的，或者说没有侧重到的。这里分享10个重要的人生必学技能，抓紧对照自己来清点一下，看看自己是不是“武林高手”。

1、战胜恐惧与接受挑战的能力

不管有多少选择项，其实仅仅就是在“过去”和“将来之间选择”。面对选择，有些人会非常纠结，评估风险、评估成本和机遇。如果你选择较简单的方案，没有风险、挑战和未知的东西，你就是选择了过去。很显然，那样会舒适安逸。但是你却永远没法实现更大的目标，除非你挑战自己。那就是为什么你必须学着选择艰难的道路，让一切恐惧你的成为你的下一个目标。

克服恐惧的能力将开启数以千计的新契机，会让你更加自信。这是学习改善人生的首要技能。如今我已为人父，同时也在努力尝试着给孩子提供满满地安全感。因为我知道，有安全感的孩子更优秀，更愿意接受挑战解决问题，更容易培养战胜恐惧和接受挑战的能力。

2、目标设定技巧

学会设定目标及分解目标，这将让你更有效的分配时间和精力资源。

规划自己的人生，通常比随意的人更成功。有人说要制定五年计划，确定大致蓝图和最终方向，但是我想说这对我们普通人来说太难了。我们不是圣人，看不到五年之后的趋势和模样，追随自己的本心作适当的修整、妥协、偏离会不会更加顺势而为？在《初探阳明心学》一文中有门徒问阳明先生：伟人们的大事业是不是都有计划？ 先生说：怎么可能有计划？他们的良知光明，守株待兔而已。

3、合理的财务管理

赚多少钱是一件重要的事情，如何聪明的规划金钱是一件更重要的事情。不管你的收入多少，你可以每月省下一笔特定金额的钱。养成省下至少20%的收入的习惯，你将有不错的积蓄。那意味着更大程度上的自由。

4、公众演说技巧

不知有多少人为此恐惧，但是它却延伸到工作及生活的方方面面。无论如何，假如你想获得自信和自尊，认识新的专业人士和社交往来，公众演说技能必不可缺。

5、批判性思考的能力

这恰恰是我们在学校里没有学到的最宝贵的技能之一。批判性思维不包含情绪的回应，而是意味着纯粹的分析和判断。它给你机会看清事实，让别人无法轻易操控你，任何时候有自己独立的思考和想法。如果你想深入的学习，可以阅读博客里之前写的《批判性思维补脑》。

6、终身学习的理念

这是我一直推崇和自诩的。因为我能看到终身学习的理念所蕴藏的巨大的能量和对一个人巨大的改变。你的人生如果比作地球，它足以翘起你的地球。毕业不是学习过程的终止，不要停止开发、进化自己。即使有一天我们的年龄很大了，也要让别人看到、让自己看到一个老当益壮的自己。

7、人际交往的能力

你的人际关系和事业，在很大程度上取决于你的沟通技巧。要学会如何与人相处、解决问题和进行建设性对话。与人交谈、交往，是理解和有效合作的关键。如果对心理学感兴趣，可以学习一下。心理学会成为人际交往中奇妙的润滑剂。

8、 学会写作

写作技巧是一个人职业生涯发展中的重点。准确、清晰、连贯的写作能力是沟通的有力工具。同时，你的想法、阅历、经验都可以随时通过笔尖流淌出来。写作是一个输出的过程，不仅仅是整理、构思，更重要的是这个过程中的沉淀、升华和内化。
