php概述

发表于 2012-03-18 | 分类于 PHP |

Php系列的笔记以这篇概述作开始。我的学习方式主要是视频教程（比如lamp兄弟连的视频），加书籍参考，再加上网上资源的查阅。自学是本人非常推崇的一种态度，整合各种资源会起到事半功倍的效果。

Php是一种可以与各种前台语言（html,javascript等）默契合作的解释型语言。Php的规范标签是<?php ?> ，这是php的官方指定规范特征。其他的简易风格跟变形标签不做介绍，这跟讨论茴香豆的“茴”字有几种写法一样没有意义。在服务器端，php解释器会将<?php和?>的代码进行解释，而不会干扰html跟javascript 。解释完后将整个页面返回给浏览器，在客户端显示出来。

看例子：

Demo.php:

<html>

<title><?php echo”PHP demo”;?></title>

<body>

<script>

alert(new Date());

</script>

<?php echo”<center>”;?>

<h1><?php echo”vat58,成长无限”;?></h1>

<p>

Vat58给你的或许就是一个成长，完善自我的动力！

</p>

<?php

For ($i=0;$i<3;$i++) {

echo”yes,I can. <br>”;

}

?>

</center>

</body>

</html>

上例看出，php真的是“无孔不入”，title标签里， body里，你想要的任何地方都可以。阅读的代码的时候不要被带晕了就好，呵呵。如果你已经迫不及待想学习php的语法，请先行一步学习。上述代码执行的结果如下图:

WordPress优化系列2加快文章收录速度

发表于 2012-03-13 | 分类于 wordpress |

建博第3天，谈如何加快文章收录的速度。

有了自己的博客，一直当作宝贝似的，工作之余一有时间就用手机上线看看，安心地看着它成长。如果你写了很多的文章，留言板上一直没有脚印，你可能像我一样沮丧，有一种被大家忽视冷落的感觉…

于是我用手机展开了探索。打开百度跟googel分别实验。一个实验的基本原则就是从用户的角度考虑问题，针对某一个搜索需求，用户可能用什么样的关键词来进行搜索。实验采用渐进式。

首先声明一点，如果直接搜索你博客里某篇文章的主题，估计想要找到自己站点的影子几乎是不可能的。因为新建的博客权重比较低，文章有没有收录还不清楚，更别说排名了。所以我们的要求不要太高。

首先搜索站点的名称vat58，百度显示的结果在第一位，google的是在三四位的样子，仅仅说明一点，站点或者说首页被收录了。然后扩展关键词，站点名称加上某篇文章的主题。比如“vat58 加快文章收录速度 ”。按照时间顺序，从写得较久的文章开始测试。如果能搜到结果，说明文章被收录了。我的结果是，google收录了大部分，百度上不是很理想，有很多没被收录。于是我开始查找资料，如何加快文章收录速度。我的收获如下。

1. wordpress的更新服务

简单的说，更新服务是当你的站点有文章或资源更新时，能够及时的通知搜索引擎，从而做到更好的收录，提高博文的曝光率，从而提高网站的流量。

在做菜单栏找到【设置】里面的【撰写】选项，找到如上图所示的区域进行编辑。第一行为默认，接下来我手动添了两行,。用来通知百度跟谷歌。更多服务列表可自行搜索。

2.适度的seo

站点或相关文章的主题要明确，包括文章中关键字的分布，文章的分类，标签，内链，外链等，都是要注意的细节，由于seo功力不足，就不班门弄斧了。

Vat58初来乍到，几乎只有自己铭记于心，哈哈。没有谁会在搜索时加上这样一个无味的关键词去青睐我的站点。所以，新建站点的曝光率还是很低的。这就需要我们新手站长持之以恒，多学习，多进步，多发表原创文章和资源，慢慢积累站点权重，才有出头走俏的一天。

关于对日开发

发表于 2012-03-12 | 分类于 天驴行空 |

毫不避讳，我现在是一名底层的对日开发人员，工作年限半年。在看本文的你或许正是涌入对日开发大军中的一员，亦或是一名资深的对日开发者，亦或是有相关的想法。下面我将谈一下我眼中的对日开发。

坐井观天，从底层的角度考虑，对日开发者是披着智慧外衣的体力劳动者。我们经过了日语培训，还是用中文的发音念日本汉字。公司从上倒下，都那么念，沟通上肯定不会有问题，这是优点。我们照着手顺，在Ctrl+C和Ctrl+V中穿梭。大家常常聚在会议室，讨论修改代码的时候，一个空格或是一个空行去留的问题，并且还要跟日方来回确认，不得不佩服这种严谨的精神。简单而又繁琐的工作，按部就班。可能会有人喜欢这种工作，我也不知道身为项目组的leader，会不会有某方面的成就感。

是的，我不是一名合格的对日开发人员，我有异心。我感觉以上的境遇滑稽可笑。没有创意，没有激情，没有实现自己想法的快感。不论从技术上，还是商业的感觉上，都不是一个很好的发展环境。我对互联网感兴趣，喜欢从用户体验的角度看待产品和服务。我喜欢交往，跟大多数人打成一片。不知道有些时候，我的非按部就班会不会被领导看作是“叛逆”。一个周末的时间，我开通了这个博客，希望能发展成一个大的站点，在这个经营的过程中，慢慢的积淀网站运营的经验，进一步努力有自己的产品上线。是不是有点遥远？或许是吧。

尽管如此，还是不要马上辞职的好。这个环境目前来看是适合我沉淀的，工作之余，我的大脑在各种想法中漂移，当然不是只想不做。以上仅代表一个对对日开发了解不深的叛逆人员的片面观点。看到这里，相信会有很多同事喷我。之前我就声明了，我是从一种坐井观天的角度考虑的，如果您是高层，就不要跟我计较了。这个职业还是为像我这样的菜鸟提供了一个缓冲适应的平台，让你有时间慢慢观察思考，进而完成角色的转变。个人认为，还是转变的好，随时保持大脑的灵活，不要被锈住啊。如果你想慢慢地走向高层，也是可以的，如果你是那样的性格。最后，将我自己总结的几句话送给我自己，作为现在工作的目标，同时献给准备进入或者刚刚进入对日开发队伍的同事。

培养一丝不苟的工作态度

处理问题有条不紊的思路

练毛笔字搬的性情和韧性

专业领域范围扫盲和概览

自我的选择性精通和拓展

windows7破解登陆密码(原创)

发表于 2012-02-15 | 分类于 Windows8 |

女朋友的公司电脑开机密码忘记了，令我相当的郁闷。怎么能如此的不谨慎呢。哎，没挨领导批，偷偷搞定已经很不错了，高手请略过本文。

待破解的系统是日文windows7，想想手边的工具不是很全,只有U盘，一张其他的系统盘。没有win PE，我的UBUNTU livecd 也没带，索性先装成双系统把。目的就是修改原系统目录里边的文件（请根据情况自行选择最简便的方法）。流行的方法是这样的：将原系统目录 …\windows\system32 文件夹下的cmd.exe跟osk.exe作一下变动。首先备份两个文件到其它地方，方便以后改回。然后，删除掉osk.exe，将cmd.exe更名为osk.exe。这样做的目的，是在登陆画面的轻松访问点击“屏幕键盘”时，可启动cmd窗口，进而进行相关帐户的编辑。

在系统目录中更改系统文件不是那么顺利，罗嗦以下我的做法吧：因为手边有360，所以我把osk.exe跟cmd.exe强力删除掉（已做过备份），也可用其它强力删除的工具实现。然后把一个cmd.exe备份更名为osk.exe,拷回system32文件夹下。OK！重启。

进入系统，在登陆界面选择轻松访问中的“屏幕键盘”，这时候就等于打开了“救命窗口”——cmd.
首先查看一下帐户信息：net user
更改帐户密码 ：net user 用户名 新密码
或者新增一个帐户 ：net user 123 /add
给123提升权限 ：net localgroup administrators 123 /add

相关的命令语法最好自行查阅掌握。
这样基本上就搞定了，就可进入原封未动的系统了。
最后不要忘记把“门”关上，恢复之前改动的文件。

总结：理论上可将轻松访问中所有操作对应的可执行文件更换成cmd,从而轻松打开cmd窗口。时间紧迫，请自行尝试。从中也可以看出，图形见面有它的好处，也有它存在的安全隐患。

今天你Google了吗？搜索技巧

发表于 2011-11-10 | 分类于 google |

“工欲善其事必先利其器”，掌握优良的搜索技巧，将对我们的学习生活起到事半功倍的效果。多一份技巧，多一个“心眼”，你淘到的就是精品。所谓搜索，就是在正确的地方用正确的工具正确的方法找到正确的内容。本文以google神器为例，整理下她的搜索技巧。

1.善于运用逻辑运算符缩小搜索范围，定位目的资讯。最基本的语法“与”“非”和“或”，这三种搜索语法Google分别用“ ”（空格）、“-”和“OR”表示。注意：这里的“ ”和“-”号，是英文字符，而不是中文字符的“＋”和“－”，“OR”必须大写。此外，操作符与作用的关键字之间，不能有空格。

2.搜索整个短语或者句子，可用英文引号。

3.搜索引擎忽略的字符以及强制搜索。搜索引擎对频率极高的关键词予以忽略。强制搜索的办法为在关键字前加上明文的“+”号。

4.用“site”限制搜索网站。限定或排除均可。例：“搜索技巧 site:www.vat58.com”

5.在某一类文件中定位资讯。“filetype:”是Google开发的非常强大实用的一个搜索语法。示例：”VBA编程 filetype:doc OR filetype:pdf”

6.搜索的关键字在URL中。“inurl:mp3 像梦一样自由” 另外，还有“allinurl:”的用法

7.关键字在标题中。“intitle:汪峰”

8.搜索所有链接到某个URL地址的网页。”link:www.vat58.com”,可以查看链接到目的网址的网站，可以通过这个手段找到自己喜欢的类似的网站。

9.查找与某个页面结构内容相似的页面。”related:www.vat58.com”,“related”用来搜索结构内容方面相似的网页。

10.欢迎补充，分享技巧
