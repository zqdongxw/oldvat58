深入解析Android UI

发表于 2014-08-20   |   分类于 Android   |  

View

所有的UI组件和一般2D图形的基础是VIEW类。这个通用类有很多的方法和属性来定义要显示的部件和图形实体，并且可以响应事件。一个Activity对应一个view对象，每一个view对象都有一个onDraw方法。onDraw可以把view对象绘在画布上。整个过程就是当需要更新UI时Activity会调用view对象的onDraw方法，一个按钮或者一个文本框，传给画布，然后渲染到屏幕上。

什么是批判性思维？
