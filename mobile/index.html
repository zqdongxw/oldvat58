未来手机终端的几点想法

发表于 2012-04-07   |   分类于 移动互联网   |     |  

1.与计算机等设备完美协同使用。比如手机可以遥控电脑，不同设备间资源的同步共享。可将电脑，手机互补推出，更具杀伤力。

2.人机合一。提高手机设备及配件与用户的粘度。将手机与个人真正绑定，一对一的认证。人机合一，获得一种拥有感。可否推出一种认证服务？将手机信息与个人信息注册认证。或者用指纹识别等技术，实现用户对手机的唯一使用“钥匙”。同时，手机辐射问题应该得到良好的处理。

3.多传感器。重力传感器，温度传感器等实现多维度拓展手机终端。

4.差异化，市场不断细分，专门的，专业的手机终端登上舞台。

5.移动互联网安全越来越受到重视。
