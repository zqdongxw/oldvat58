Ubuntu14.04 力荐Fcitx输入法

发表于 2012-06-04 | 分类于 ubuntu |

在之前的博文中介绍了IBUS中的谷歌和日语输入法，详见《ubuntu12.04 谷歌+日文输入法》。这次着重推荐小企鹅输入法Fcitx及它的云拼音支持。已笔者之见，Fcitx用户体验更好，输入更智能流畅。

Fcitx 4.2.0版较老版有了很大的改进，内置了GOOGLE拼音。

删除ibus输入法：sudo apt-get remove ibus
安装FCITX： 
sudo apt-get install fcitx fcitx-pinyin fcitx-sunpinyin  fcitx-table

需要重启。

为Fcitx增加云拼音的支持

 sudo aptitude install fcitx-module-cloudpinyin #云拼音模块支持     //现在好像不能用了。


云拼音来源可选择搜狗、google、百度等。体验如下：

在配置界面，默认的最小触发云拼音的拼音长度为10,大家可以自行改小，使搜索结果更“云”化，更智能化。上图可以看到，第二项为云提供的结果，给力！
