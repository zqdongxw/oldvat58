Axure元件库推荐

发表于 2018-03-30 | 分类于 axure | |

在github上发现一套丰富的Axure元件库，献给axure新手上路。

基于Axure RP 8，支持 Android、Apple、Windows、微信，移动、桌面平台的应用和网站原型设计。

如何安装元件库

Axure - 「库」 -「载入元件库」，选择下载的 .rplib 的文件即可。

元件库内容

☞ Material Design
基于 Google Material Design 准则创建，适用于 Android 手机、平板应用，以及跨平台桌面应用的原型设计。

☞ Apple Design
基于 Apple iOS/macOS Human Interface Guidelines 创建，适用于 iPhone、iPad 和 macOS 桌面应用的原型设计。

☞ Fluent Design
基于 Microsoft Fluent Design System 创建，适用于 Windows 手机、平板和桌面 UWP 应用的原型设计。

☞ 移动网页
基于微信 WeUI 和小程序设计指南等创建，适用于包括微信在内的移动网页或 HTML5 轻应用的原型设计。
预览

☞ 桌面网页
包括通用官网，社交、电商和媒体等分类网站和控制台的设计元素。

☞ 模型和数据图表
模型图表，包括描述产品需求流程的 UML 状态图、活动图、序列图、用例图，以及商业等分组图表。

数据图表，包括适合移动端的点图、线图、区域图、柱状图、饼图、雷达图、股票图、仪表盘及混合等各类数据图表；

☞ Material Design 图标
基于 Material Design icons V1.0.2 图标创建，包括 action, alert, av, communication, content, device, editor, file, hardware, image, maps, navigation, notification, social, toggle 等分组图标。

☞ 品牌设计图标
基于 Apple、IBM 等品牌图标创建，包括 Apple 系统导航和工具栏、切换栏、快捷操作，IBM Design 操作、格式、对象等分组图标。

☞ Icons8 图标/图片
基于 Icons8 图标创建。

☞ 图标平台图标/图片
基于 Easyicon、Smallicons、阿里妈妈 Iconfont 等图标创建。

☞ 手势图片
基于 Jeff Portaro 的 Touch Gestures Icons 创建，适用于所有移动平台触控、桌面触控，以及触控面板的操作手势表达。

☞ Font Awesome 字体图标
基于 Font Awesome V4.7.0 图标字体创建，在使用前，需要在操作系统设计中先安装此字体 FontAwesome.otf。
