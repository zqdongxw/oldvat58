Mysql小记

发表于 2012-05-24 | 分类于 数据库 |

说到mysql数据库，中小型够用，开源，比oracle轻量，得到不少人的青睐。在此记录一下mysql的常用操作。

Java中连接mysql数据库（jdbc）

官方获取mysql驱动程序包
Class.forName(“com.mysql.jdbc.Driver”); //加载驱动程序，”org.gjt.mm.mysql.Driver”的写法兼容但不建议使用。
Connection conn =
DriverManager.getConnection(“jdbc.mysql://host/DBname”,”用户名”,”密码”);

String sql = “select * from users where count=? and name=?”;
PreparedStatement ps = conn.preparedStatement(sql);//据说用PreparedStatement档次高点
ps.setInt(1,int1);
ps.setString(2,var2);
ResultSet rs = ps.executequery ();

Mysql常用指令

进入mysql数据库控制台
mysql -u root -p dbname; ［回车输入密码］
mysql> show databases; [查看数据库]
mysql>create database <数据库名> [创建数据库]
mysql>use 数据库; ［选择数据库］
mysql>show tables; [查看表]
mysql> create table pet (name varchar(20) primary key not null, owner varchar(20),
-> species varchar(20), sex char(1) default ‘0’); [创建表及相关属性]
mysql> describe pet; [显示表结构]
mysql> select * from MyClass order by id limit 0,2; ［查询前两行数据］
mysql> alter table MyClass add vol int(4) default ‘0’; ［增加字段vol]
mysql> alter table table_name rename new_table_name; [更改表名]

alter table 表名称 change 字段原名称 字段新名称 字段类型 [是否允许非空];
alter table 表名称 modify 字段名称 字段类型 [是否允许非空];
