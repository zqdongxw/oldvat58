wordpress到底火了多久？我落后了多久？

发表于 2012-04-15   |   分类于 wordpress   |  

至今，vat58还是一个婴儿，将近一个月了。左看右看，依旧有很多不尽如人意的地方。比如，页面不是很美观（看到很多博客都有那种淡淡的模块边框，有板有眼），目前只有“首页”跟“about me”两个页面（新建的页面用什么内容填充还是不清楚），诸多困惑。


你看，上图中“发表于…由…”这种格式有没有很丑？posted on…by…这样还可以接受，中文版的wordpress内部的翻译机制没太搞明白，主题文件夹下的languages里面的zh_CN.mo和zh_CN.po文件是如何工作的？一时半会儿是搞不清楚了。魅力指数这个字有点大？

今天花了相当长的时间去研究wordpress的模板文件，尽管我的php基础薄弱。从网上找了很多关于主题修改的资料。大部分都是“过时”的，年份都很早了。之所以说“过时”，是对于“生搬硬套”来说的。从很多资料的日期来看，是四五年之前的，估计那个时候我还在读高中吧，可是wordpress已经在流行了。至今wordpress跨了N多的版本，有些教程可能已经不那么好用了。可能很多朋友犯了跟我一样的错误，喜欢拿过来直接用，要是按照教程一步一步的做能搞定最好不过了。可是事实证明：生搬硬套是要犯错误的，参考资料自己研究掌握“实情”与时俱进才是好同志！这应该适用于任何东西的学习吧。

wordpress火了很久，我也落后了很久。一些事情总是后知后觉。至少有了这个博客，可以一直督促我学习，记录，总结。可是给自己提个醒，不要为了写而写。

为了给上图中的问题一个交代，我研究了functions.php里面的twentyten_posted_on函数：

function twentyten_posted_on() {
printf( ( ‘<span class=”%1$s”>Posted on</span> %2$s <span class=”meta- sep”>by</span> %3$s’, ‘twentyten’ ),
‘meta-prep meta-prep-author’,
sprintf( ‘<a href=”%1$s” title=”%2$s” rel=”bookmark”><span class=”entry-date”>%3$s</span></a>’,
get_permalink(),
esc_attr( get_the_time() ),
get_the_date()
),
sprintf( ‘<span class=”author vcard”><a class=”url fn n” href=”%1$s” title=”%2$s” rel=”author”>%3$s</a></span>’,
get_author_posts_url( get_the_author_meta( ‘ID’ ) ),
esc_attr( sprintf( ( ‘View all posts by %s’, ‘twentyten’ ), get_the_author() ) ),
get_the_author()
)
);
}

还是再去学习下php吧。
