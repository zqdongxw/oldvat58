安卓手机刷入google组件

发表于 2013-05-30   |   分类于 Android   |  

本人非google重度用户，接触网络从大学开始，接触google的时候她已经退出大陆了。但是对于google的大部分产品，自己真的是非常喜爱。

google的产品越是优秀，大陆的用户就越纠结，总有个什么东西牵绊着与google的连接，你懂的。

安卓在google中扮演着重要的角色，如今在智能手机领域占据了大半江山。可是国产的安卓已经阉割掉了谷歌组件，可惜了那些手机版的优秀产品。

本篇文章的主题是如何在安卓手机正常使用安卓的应用，周边的话还有翻墙等等。

之前自己摸索的时候，chrome里登陆账户一直没有反应。查了下资料说是缺少谷歌服务框架等组件。于是找了个几个分别安装，网上也有教程。可是一直没成功，系统时而卡住，谷歌账户也添加不了。所以还是建议大家直接刷入谷歌组件，干净利索。

本文章的基础是手机已经root并且刷入recovery。这方面就不在赘述，网上很多教程。手机是联通定制版9300.

1.下载谷歌服务包, 搜索gapps-gb可以找到很多服务包,大家可以根据包含的组件进行选择.

将服务包放到手机存储卡上等待刷入

2.手机先关机状态,然后同时按住音量上+Home键+电源键 进入recovery模式,选择 install zip from sdcard

找到之前放好的服务包刷入.

3.重启 ok

这样一来,谷歌服务框架、google play商店、google+、google talk 等等 已经收入你的爱机了。实际上，很多应用可以自行下载安装。

没有了闪屏退出,之前的chrome也可以登录账户，同步pc端的书签。太棒了。

[caption id=”attachment_699” align=”alignleft” width=”168”] google+[/caption]

目前好像手机端不用翻墙就可以使用google+等应用，除了youtube. 个人感觉google+改版后更适用于移动终端。

还没完, 当我兴冲冲从play商店下载应用时又碰到了问题：下载完毕后提示可用存储空间不足。

而事实是还有大把的空间可用.

[caption id=”attachment_698” align=”alignnone” width=”168”] play商店[/caption]

而刷入google组件之前,这种情况从没出现过.

于是我从recovery模式下进行了双wipe,问题解决。或许在刷入谷歌组件前，应该双wipe一下。
