rework

发表于 2016-05-27 | 分类于 书摘 | |

很久没有读到像rework（重塑工作）如此富含营养的书籍了，希望它也能够开拓你的视野，在你枯燥的工作生涯里注入一股清泉。以下为《rework》书摘：

小、简单、基础的需求是恒久的

1.真实的世界不是一个地方，而是你一个借口，是为毫无尝试而辩护。

所以如果某人说，你这个想法在真实世界里是行不通的，先别急着泄气，好好盘算一下，试试就知道了。

2.决定你这个星期该做的，而不是几年。弄清下一件重要事情，然后去做。在你做某事之前找出正确的方向，而不要过于超前。

我一直头疼于五年计划之流。

3.每个人都应该被鼓励开创自己的事业，而不是一些自我认同为创业家的稀有品种才去创业。

让创业走向神坛，好好的打磨自己某一方面的技能，然后去做点事情。

4.解决你实际遇到的问题会让你爱上你做的事情。你知道问题所在并且熟知解决它的价值。这是无法替代的。

看，这就是创业的素材和灵感源泉，不要一开始想着市场需要什么，先从你自身入手。如果你开公司是为了满足别人的需求，那就搞错了。很新颖的观点。

5.质量的增长是因为：长期合同、多余的员工、固定的决策会议、厚重的流程、库存、硬件、软件、技术锁定、长期路线图、办公室政治

质量越大，越不灵活。短小精悍灵活才是本书所提倡的。

6.你要从你不得不去做的事开始，从核心开始。

7.时尚会褪色。

8.人们对于工具的热情远大于依靠工具所做的事情本身。

9.如果你想描述什么，写实一点。不要描述一件事物像什么，应该画下来。不要描述一件事物听起来怎么样，应该哼出来。做每一件事情都少一点抽象描述。

怎么样，是不是干脆利落具体？

10.做一个反对某某者是区别他人，吸引别人注意的好方法。

国内的各行各业的大佬都精通此法，尽管是在模仿。

11.比你的竞争对手做的少来打败他们。把麻烦的、艰苦的、讨厌的问题留给竞争对手。不要去胜一筹，去超过别人，试试想法的做法。

又一个不可思议的观点。轻装、简洁、专一。

12.突然一天大客户走了，公司会背负一个包袱—这个产品是围绕一个已经离开的人设计的，而其他人没法用。

13.我们所希望的是顾客因为我们的产品而成长，绝不是我们优先去适应他们。为有些人增加一些高级用户的功能会威胁到一些还不是我们用户的人。

14.小、简单、基础的需求是恒久的。

15.不用写下用户的需求，也不用担心遗忘，真正的需求你会不断的听到。忘掉的就算了。

16.与其出去找顾客不如让顾客送上门来。拥护者会前来找你，主动的，关心你说什么。这是你所遇到的最能接纳你的、潜在的用户。

17.展示你的后台，让人们走进幕后可以改变你们之间的关系。他们会觉得和你有一种联系，看到你人性的一面而不是一个没有个性的公司。他们会看见你产品里所包含的汗水和智慧。

18.不要惧怕表露你的瑕疵，不完美是真实的，也是人们乐于接受的。这就是为什么人们喜欢凋谢的真花，而不是塑料花。

19.真正的营销是你所做的所有事情的集合，而不仅仅是销售部门。

20.在小的团队里，人员是来工作的，而不是来发号施令的，每个人都要做事，不能凌驾于工作至上。

21.如果你要从一堆人中决定出一个职位的合适人选，雇那个写作最厉害的人。

22.最糟的道歉方式之一是不含道歉的道歉，听起来是在道歉，但实际上没有真正的承认过失。好的道歉要承担责任。

比如常见的：我们为对此可能带来的任何不便感到抱歉。这完全是不负责任的套话，它忽略了客户所遇到的真正的问题。

23.团队里每一个人都应该和客户有联系。

24.当所有事情都不断的需要批准的话，你则培养了一种不动脑子的文化。

25.如果你禁止员工在工作时说话，访问交际网站或者观看youtube，你能得到什么?什么都得不到。

什么？什么是youtube? @!！

26.你不可能让人8个小时都在工作，那是神话。
