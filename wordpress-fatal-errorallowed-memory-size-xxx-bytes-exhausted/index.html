发表于 2014-10-06   |   分类于 wordpress   |  

wordpress fatal error:Allowed memory size of xxx bytes exhausted

解决方案：

修改wp-config.php文件：

查找：define(‘DB_COLLATE’, ‘’);

在下一行添加：define(‘WP_MEMORY_LIMIT’, ‘64M’);

如果没能解决问题，可以尝试更大的值，比如128M.
