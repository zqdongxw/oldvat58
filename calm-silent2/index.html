袖手无言味最长(二)

发表于 2013-06-24 | 分类于 V文摘 | |

袖手无言系列内容大多来自本人的读书笔记、思考以及网络中一切乐于跟大家分享的内容。在@VAT58博客你也许会发现不少影子。

  1. 【你的灯亮着吗】问题的本质是现实体验跟期望之间的差距。解决问题前，先要学会思索这是谁的问题。你的灯亮着吗？转换问题对象有时候可以找到更好的解决办法。

  1. 【解决问题原则】当别人能很好的解决自己问题的时候，千万不要越俎代庖。这个时候的热心是一件不恰当且画蛇添足的事情。还有一句话：如果你不能解决问题，你就会成为问题。

  2. 【对数学的滥用】钟形曲线（正态分布）容易描述自然问题，它信奉的是“损有余而补不足”的天之道，平均主义；幂律曲线（或许你听过二八法则）容易描述社会问题（比如金钱），它信奉的是“损不足以奉有余”的人之道，极端主义。但是道可道，非常道，没有任何一个定律是放之四海而皆准的。

  3. 【酸葡萄甜柠檬】吃不到的葡萄是酸的,这是酸葡萄心态;吃到的柠檬是甜的,夸大已经得到的柠檬的价值,这是甜柠檬心态. 比如: 张三之前爱讲:好男不当兵,好铁不打钉. 阴差阳错服兵役后,开始宣扬: 男人一生一定要当次兵。

5. 成功学的思路是,先提出一个论点,然后寻找证据.正如一个律师,不会为自己的当事人网罗不利证据.他们对相反的证据视而不见.成功学的本质是把小样本中的特征强推到大样本中. 大的成功,多是基于运气. ——《赌客信条》

6. 记得之前上学时，总会有人说我谦虚，最后还得警告我过分的谦虚就是骄傲。现在培根告诉我：谦逊，通常是自负者欲扬先抑的诡计。

  1. 绝对理性实际上是一种非理性.选不到最优,就选次优,这才是最超越理性的超然行为模式. 太过理性、纠结必然导致决断力崩溃。

  2. 夏娃: 亲爱的,你会全心全意爱我吗? 亚当:这是肯定的.除了你,我还能有什么选择呢? 少则得,多则惑.是以圣人抱一为天下式.

  3. 记得小时候吃鸡蛋下面条，我总是把鸡蛋留在最后吃。现在我知道这叫做推迟满足感，是自律不可缺少的一个原则。试着去推迟满足感，你会发现自己愈发地自律、有魅力。

  4. 如果你半夜醒来发现自己已经好长时间没读书，而且没有任何负罪感的时候，你就必须知道，你已经堕落了。不是说书本本身特了不起，而是读书这个行为意味着你没有完全认同于这个现世和现实，你还有追求，还在奋斗，你还有不满，你还在寻找另一种可能性，另一种生活方式。—— 陈平原

  5. 世间没有后悔药，“早知如此”这句话里透漏了懊悔者满满的遗憾无奈。改变造成的懊悔称之为行动的懊悔，忽视改变造成的懊悔称为忽视的懊悔。而往往前者总是大于后者。出于对懊悔的规避，所以人们才会安于现状，墨守成规。勇于改变的前提是要有良好的心理卫生。

推荐上文提到的书籍：《你的灯亮着吗?:发现问题的真正所在》 《赌客信条(钻石版)》
