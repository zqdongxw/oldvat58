Android学习笔记系列06–ListView

发表于 2012-05-30   |   分类于 Android   |  

Android中ListView控件使用起来相对复杂，我们着重来学习一下。例子的效果是实现一个ListView，包含图片，标题，内容。演示效果如下：

本例中layout有两个配置文件
main.xml （定义ListView）

<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android&quot;

<TextView

private List&lt;Map&lt;String,Object&gt;&gt; getData() {

在ListView中，SimpleAdapter是一种扩展性非常好的适配器，是需要着重理解的一个概念。它可以将静态数据（图片，文本等）映射到xml所定义的视图上，是数据映射到ListView的中介。

SimpleAdapter (Context context, List<? extends Map<String, ?>>data, int resource, String[] from, int[] to)

参数：
data：映射列表
resource：列表条目试图的布局文件
from：可以理解为列表中的列名称
to：可以理解为列名称 对应的内容
