Androidify you --定制你的Android机器人

发表于 2012-05-05 | 分类于 移动互联网 |

来看一款Android手机软件：Androidify Androidify是由Google谷歌官方开发的Android平台的虚拟形象软件，你可以在android手机上安装Androidify，自定义安卓机器人的肤色、发色、衣服、裤子、鞋子、饰品等方面，还可变更身体的比例，把Android机器人的头部、身体、四肢进行随意的变换，拉长或者缩短，甚至可以加上一款时尚的Android手机，做成与真人相似的卡通机器人，在手机上创造属于自己的Android形象。

如果你已经在自己的手机上摆弄起来，那就不打搅你了。如果你没有一款Android手机，也不要着急。还记得前几天我们搭建的ubuntu12.04+android sdk+jdk+eclipse环境吗？这才是本文的目的所在。

接下来给android虚拟机avd安装Androidify.apk

  1. 启动AVD
    emulator -adv avdname 或者 emulator @avdname
  2. 安装apk文件
    adb install Androidify.apk
    command not found?

    如果碰到上述问题，说明你还没有配置相关的环境变量，这里算是ubuntu android 开发环境配置的补充。
  • 编辑bashrc文件，加入android-sdk目录下的tools和platform-tools路径
    gedit ~/.bashrc
    在文件最后，添加： export PATH=<yourpath1>:<yourpath2>:$PATH
    保存退出。
  • 使配置生效
    source ~/.bashrc
    linux环境下你能搞定，windows当然就不在话下了。之后进行上述启动AVD，安装apk文件的操作。OK, 看下我的文艺android机器形象。
