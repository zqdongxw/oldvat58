儿时记忆

发表于 2012-06-01 | 分类于 V文摘 |

又是一年的儿童节，这是一个全民的节日。成年的我们年龄在增长，儿童节就有了双重的意义：陪下一代的儿童过好儿童节，安顿好自己的回忆和一颗未泯的童心。

你懂得

儿时画的丁老头，还是傻傻的，记忆里依然没有老去；
儿时折的纸飞机，哈一口气，一直飞在童年的时光里；
时光偷走了我的玻璃弹珠，却夺不去一颗未泯的童心。
课间十分钟，骑马打仗的“战场”，曾经“硝烟弥漫”，热火朝天…

不知道

丑小鸭的故事还在上演吗？
卖火柴的小女孩，在天堂跟奶奶过得可好？
海尔兄弟找大后还是那么聪明吗？
教室里刻着“早”字的书桌去了哪里，我的历任同桌们你们过的还好吗？

童年有六个一千零一夜，每一夜都有一个童话与我们一起入睡。现在的孩子玩什么呢？当今媒介丰富多彩，手机、pad等设备大肆横行，电脑游戏占了很大的比重。传统意义上的纯真时代已经不在，“杂质”多了，成长还会好吗？当下有个名词叫儿童成人化，这种“早熟”是扼杀孩子们的童年周期吗?曾经屁颠屁颠的80后，现在也到了为人父母的年龄，祖国的花朵需要我们更好的呵护。
