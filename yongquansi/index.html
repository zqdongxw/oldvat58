涌泉寺游记

发表于 2018-05-20 | 分类于 游记 | |

这几天为出差密集期，先到广州，中到上海，后又到福州。第一次到福建，想起了《百年虚云》。虚云老和尚出生于泉州，后在福州涌泉寺出家，于是福州之行就有了厚重的期待，参访鼓山及涌泉寺。

鼓山得名于其山顶上一块酷似大鼓的岩石，且有传说每逢风雨之际，就会有鼓声由上中岩洞传出，因此有鼓山之名。涌泉寺在鼓山山腰。山上有涌泉寺、喝水岩、十八景、观音阁等自然和人文景观。鼓山上还有古代摩崖石刻多三百多处，其中有蔡襄、朱熹等人的手书，是福州古代石刻最多最集中的地点。

当地气温39度，进此山门之前，远未想到鼓山之行如此美好，如此清凉。

南方多榕树，北方很少见。

山上环境清幽清凉，山路走几步就气喘吁吁。同行之人，男女老少，音乐律动不绝于耳，助力登攀。中年大叔上身裸露，汗珠细密，走走停停，不敢久憩，生怕落后。我刚开始还有点矜持，后来…


800m的路程。


1200m的路程。

鼓山上古代摩崖石刻多众多，手机拍照已经停不下来，慢慢忘记了疲惫。越接近涌泉寺，氛围愈来愈庄严。

百善孝为先，万恶淫为首。谨记。

南无阿弥陀佛。

作如是观。

山中一隅。想到虚云禅师曾经于山中修行三年，出家真乃大丈夫之所为也，心不坚做不了出家人。

山路减缓，离涌泉寺越来越近。寺院与幽山，天作之合，如此的理所当然。
