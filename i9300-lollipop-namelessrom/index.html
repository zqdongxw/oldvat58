i9300刷入Android5.0-lollipop之NamelessRom（保持更新）

发表于 2014-12-02   |   分类于 Android   |  

首先评价下GT-i9300吧，这款曾经的旗舰明星机型到现在依然宝刀未老。入手i9300两年有余，如今屏幕上多了几处裂纹，之前也换过一次显示屏，不得不感叹智能手机屏幕的硬伤，屏幕依然是最容易坏的部件。时间会在硬件上慢慢留下痕迹，我也慢慢对硬件兴趣减淡，或许只有刚入手的时候才有那份独有的新鲜感。机器上的软件核心系统，谷歌服务框架、G+、绿色守护、Kindle APP、Google Keep、Evernote、Ted 等各类应用软件构成我工作生活的整个手机生态系统才是我对手机的依赖所在。给我任何一台新的机器，没有任何吸引力，因为没啥可用。因为活跃的社区、基友、丰富的第三方rom支持，使得I9300的可玩性依旧很高。

在《i9100G体验lollipop》中已经体验过android 5.0 ，CM的lollipop依旧不见踪影，最终决定尝试下NamelessROM，不能再等了。

1. 资源准备

nameless-5.0-20141128-i9300.zip (链接: http://pan.baidu.com/s/1i3BrMpJ 密码: 4nho)

nameless-5.0.2-20150101-i9300.zip （官网地址） / 20150102更新 /

已刷入第三方recovery，比如CWM、TWRP，并更新到新的版本，如何刷入recovery? 参考《ADB指令刷入recovery、获取root权限、刷机》

2. 如有必要备份现有的手机系统，以备不时之需。

3. 双清: wipe data / factory reset + wipe cache partition ，然后刷入rom

4. 刷入Gapps(可选)（链接: http://pan.baidu.com/s/1pJkboc7 密码: mo27）

20141128：自带相机无法使用，使用google相机代替；无法使用移动数据，可刷入补丁解决；耗电严重。

20141221：耗电问题得以改善，移动数据修复，自带相机问题修复；从应用切换至桌面时出现卡顿，应该与5.0的内存泄漏bug有关.
