ubuntu有关权限

发表于 2012-05-14 | 分类于 ubuntu | |

如果是简单的上网、办公，ubuntu同样简单上手，流畅便捷；
如果你是linux新手，在涉及到一些高级应用的时候就可能磕磕绊绊。
不要沮丧，我们要做的就是逐步消灭每一个障碍。这一篇是ubuntu有关权限的小总结。

在shell中，可以用 ls -l 来查看目录下文件的详细信息，包括权限。

上图中第一串字符对应的是文件的权限。总共十个字符，我们一点点解析下。
第一个字符：代表文件的类型。比如 d 代表目录，- 代表一个档案（“文件”）。
接下来的9个字符代表三组权限，是 r w x - 的组合，分别对应文件的拥有者、群组、其他成员的权限。
其中，r w x 分别表示可写，可读，可执行；- 表示没有对应位置上的权限。权限可由二进制表示，比如rwx用111表示，r–用100表示。
可见，rwx=4+2+1 =7 r–=4+0+0=4
drwxrwxrwx 对应的权限就是777，即拥有者，群组，其他成员都拥有相同的可读可写可执行的权限。

如何改变文件的权限？

指令：chmod
方式一：数字类型
chmod 777 filename
如果要级联子目录的话，加参数 -R。

方式二：符号类型

chmod

u代表user(拥有者)，g代表group(群组)，o代表others(其他)，a代表所有。
比如在设定权限-rwxr-xr-x时，可以写作：
chmod u=rwx,go=rx filename (g、o合并写了)
