WordPress完善笔记

发表于 2012-03-16 | 分类于 wordpress |

每一天，vat58都在于您一同成长。

本笔记长期更新。

13/02/04

插件升级错误：

wordpress从后台升级插件时，有时候出现升级错误导致的假死，时间过长等情况。

这个时候刷新页面会提示：正在执行例行维护，请一分钟后回来。

解决办法：删除主机空间根目录下面的 .Maintenance 文件。

12/06/07

loop-single.php中文乱码，问题没有找到。
安装add-post-url 代替手动添加的版权声明。

12/03/16细节修正点：

  1. 增加文章浏览次数的功能
    小编是通过postview插件来实现的。插件安装完后，在loop-single.php中文章标题对应的代码后面加入一下代码：
    <?php if(function_exists('the_views')) { the_views(); } ?>

2. 安装Google Analytics

谷歌推出的这项分析服务，对于站点的建设和发展有着非常大的帮助。各种分析和统计对于站长进行下一步的优化提供了方向，因此是站长必备工具呢。

首先注册Google Analytics，按提示进行，一直到拷贝代码的地方。小编将拷贝的代码复制到主题目录下footer.php中</body>之前的位置上，完成了Google Analytics的安装。尽享她的奢华服务吧。
