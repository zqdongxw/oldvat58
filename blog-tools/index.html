hexo博客的几个工具

发表于 2018-02-28 | 分类于 hexo | |

本博从2015年就从wordpress转换到git+hexo的搭建方案，由于一年多的时间没有更新博客，很多操作已经逐渐生疏。翻了翻之前的笔记记录wordpress迁移到git+hexo。本博采用的是NexT.Pisces主题，感谢作者及贡献者一直以来对NexT的优化，让她越来越漂亮优雅。通过主题的使用文档，小白也可以对主题以及第三方服务进行配置，满足博客搭建者各方面的需求。在这里，博客的搭建跟拼插积木一样乐趣无穷。

这几天主要做了这么几方面的完善：

图床

选择微博图床作为博客图片的容器。在chrome中安装「新浪微博图床」插件，就可以上传管理图片资源，自动生成markdown语法，轻松引用到文章中。

音乐云盘

在写上一篇文章心经中，很想将一首古琴演奏版的心经引用到文章中，供大家欣赏，于是研究了一下当下可以用于外链的音乐云盘。如果你想上传自己喜欢的音频资源并生成博客的外链，推荐opendrive,免费5个G,也可以上传图片作为图床。目前使用不错。如果你有更好的，也欢迎分享给我。

评论系统

没有评论系统，博客终究不是一潭活水。我目前使用的是hypercomments，已经集成在next主题中，按照官方文档注册配置即可，当然也可以选择其他的第三方服务。我所看重hypercomments的是支持匿名评论，省去了登录的麻烦。

暂时写到这里，本文会根据实际体验更新，也欢迎网友的建言。
