ubuntu下三国杀本地存储允许按钮失灵解决办法

发表于 2014-10-24   |   分类于 ubuntu   |  

UBUNTU下的flash问题一直令人非常讨厌，首先表现在安装ubuntu额外版权受限程序时等待安装flash的时间长得让人发指。这个问题的解决方案是单独安装FLASH：

sudo apt-get install adobe-flashplugin adobe-flash-properties-gtk

这样一来ubuntu下视频播放的问题应该可以解决了。

至少目前来看，Linux环境从来不是一个理想的游戏环境，可是玩点网页游戏不过分吧，ubuntu下用firefox玩三国杀又碰到了问题，元凶还是Flash。当网页版三国杀运行到请求本地存储界面时，Flash对话框中的“允许”按钮就失灵了，即便你在前面的时候已经安装了flash。经过苦苦搜索，分享一个简单的解决办法。

打开：http://www.macromedia.com/support/documentation/en/flashplayer/help/settings_manager07.html

我们可以在此页面里对三国杀站点的本地存储进行设置。如下图：

重启firefox，开杀吧！
