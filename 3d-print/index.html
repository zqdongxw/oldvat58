畅想3D打印

发表于 2014-10-05 | 分类于 天驴行空 | |

谈到3D打印，先来介绍一个人，克里斯.安德森 。主要著述有 《长尾理论》《免费》《创客》 。在不到50岁的时候，他完成了职业生涯的一次大转型：离开互联网 投身3d打印， 呼吁第三次工业革命，认为3d打印是一件比互联网还大的东西。

什么是3d打印？简单的讲，基于互联网和一系列的材料技术，计算机根据设计文件控制3d打印机，由喷嘴一层层的堆叠材料形成实体。有一个时髦的名词：增材制造。跟传统工艺不同，3d打印做的事加法运算。同样由数据远程驱动制造，普通的打印技术（2d），可以实现电子版跟纸质版的转化，带给了我们很多的便利。3d打印，仅仅多了一个维度，所带来的创新和革命确是指数级的爆炸增长。

工业社会的终结者。工业社会的两大特征：标准化大规模制造，精细的人员分工。枯燥的流水线作业，将人的效能压榨到秒，才能体现工业化的优势。工业社会缔造了消费者跟生产者的大分裂，批量化标准品抹杀了个性化需求。解决这个矛盾的一个方法就是利用媒介向大众打入要营销的需求理念，固化人们的需求，从而实现批量化的需求。但我们还是清楚，只有量体裁衣才能真正合适。有了3D打印，通过设备扫描获取数据，加工形成设计文件，然后接驳到就近的3d打印机打印即可。是啊，整个互联网的思维就是去中心化，实现分布式的生态系统。

互联网很重要，有一个鸿沟需要由3D打印来填充。目前，生活的方方面面都跟互联网有着千丝万缕的联系，虚拟空间已经完善，物质生产怎么办？虚拟与实体世界之间的鸿沟，由3D打印来解决。数据远程驱动制造，真正实现由比特和原子两个世界的组成基石之间的转换。今天的3d打印机已经可以打印形形色色的材料：从尼龙到玻璃，从巧克力到钛，从水泥到活细胞。不同的材料，不同的领域和行业，一旦与3D打印相撞，就容易产生爆炸式的指数增长创新。未来有一天这样的场景就会司空见惯：请把我的鞋子传真过来。总有一天，我们编辑物理世界会像编辑数码照片一样容易。这种虚拟世界与实体世界经络的互通，只有靠3D打印，尽管还有很多需要突破的技术难题。

RepRap是一种三维打印机原型机（或3D立体打印机），它具有一定程度的自我复制能力，能够打印出大部分其自身的（塑料）组件。这就意味着机器可以创造机器，机器的未来也许就是生物，用生物学的模式重构了机器。不由得想起KK的失控，机器最终会超越人类，人类世界总有失控的一天。
