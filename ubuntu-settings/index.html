ubuntu12.04环境配置（翻墙，jdk，eclipse，Android）篇

发表于 2012-05-03   |   分类于 ubuntu   |     |  

一：翻墙

ubuntu下的chrome速度大大的优于windows下的龟速，这个确实很赞啊。真正要做到畅行，呵呵，还得翻。windows下的教程详见《不会翻墙？GoAgent带你畅游博客中的博客》
ubuntu下的方法大同小异。

配置：local目录：修改proxy.ini 中appid为你的ID
server目录：python uploader.zip （终端命令），按提示输入appid，邮箱，密码等。
运行： local目录：python proxy.py（终端命令）,接下来配合chrome switchysharp就可以成功翻墙了。浏览器端的配置同windows下。

二：截图工具

shutter: sudo apt-get install shutter 或者从软件中心查找
系统自带的screenshot也不错。
print screen 和 alt+print screen 等快捷键，在条件不苛刻时是最便捷的截图方法。
作为一名伪站长，截图工具不可少啊。snagit,抽空回windows看你。

三: openjdk+eclipse

安装openjdk: sudo apt-get install openjdk-7-jre 不知道这个版本是不是稳定。
安装eclipse: 从eclipse官网下载 Eclipse IDE for Java EE Developers for linux
解压即可。

四：Android开发

 

为eclipse安装ADT插件：
运行Eclipse，选择help->Install new software 选择add，将会弹出一个框 ：
Name: 自己随便取一个
Location: http://dl-ssl.google.com/Android/eclipse
安装developer tools。
重启eclipse,Window->reference，设置Android SDK Home。
在eclipse工具栏，找到AVD manager创建AVD。
至此，Android开发环境完成。

拓展阅读：定制你的Android机器人 鸟哥的Linux私房菜:基础学习篇(第3版)
