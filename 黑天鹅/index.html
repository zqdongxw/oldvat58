黑天鹅

发表于 2015-05-27 | 分类于 心理学 | |

黑天鹅是不可预测的重大事件。一只黑天鹅的出现打破了人们看似不可动摇的白色认知。人们习惯于用自己有限的生活经验和不堪一击的信念来解释这个复杂的世界，结果被一只只黑天鹅打得遍体鳞伤。克里斯.安德森说：对于世界的运行方式，《黑天鹅》揭示的内容比十几座图书馆加起来还多。一起来了解一下。

作者的观点：我们的世界是由极端、未知和非常不可能的事物主导的，而我们却一直把时间花在讨论琐碎的事情上，只关注已知和重复发生的事物。

作者做交易员时候的自我修炼：只关注最技术性的方面，从不参加商业会议，避免与西装革履却不读书的“成功者”为伍，每三年休息一次，弥补自己在科学和哲学方面落下的空白。为了慢慢提炼思想，成为一个闲人，一个职业冥想者，懒洋洋地坐在咖啡馆，远离办公桌和其他组织，睡到自然醒，贪婪地阅读，逐步在黑天鹅思想基础上构建一套完整的思想体系。

##极端斯坦与平均斯坦

属于平均斯坦的问题举例： 身高、重量、面包师的收入，不可能有突破性的变化。
输入极端斯坦的问题举例： 财富、收入、JK.罗琳的图书销量，从事极端斯坦的投机，可以在一分钟之内就赚取或赔掉大笔财富，并且大部分黑天鹅现象发生在极端斯坦。

##从火鸡的经历中学习

哲学家罗素在阐述所谓的归纳问题时举了一个绝妙的关于意外的例子。
从观察获得的任何知识中都有陷阱

想象一只每天有人喂食的火鸡。每次喂食都使他更加相信生命的一般法则就是每天得到友善人类的喂食。感恩节前的下午，一件意料之外的事情发生在它身上。对它造成一次信念的转变。随着友好喂食次数的增多，它的信心也增加了，虽然被屠杀的危险越来越近，它却感到越来越安全，当危险最大时安全感却达到最大值！真正的问题比这更具有普遍性，直指经验本身。某种东西过去一直起作用，直到它出乎意料地不再起作用，而我们从过去获得的知识实际上是虚假的，甚至是危险的误导。你又怎么知道你不是那只火鸡呢？！

关于对过去经验认识的问题举例，让我们来看一下那位著名船长的表述：

根据我所有的经验，我没有遇到任何值得一提的事故。我在整个海上生涯中只见过一次遇险的船只。我从未见过失事船只，从未处于失事的危险中，也从未陷入任何有可能演化为灾难的险境。

——史密斯，泰坦尼克号船长，1907年

史密斯船长船长的船沉默于1912年，成为历史上被提起次数最多的沉船事故。

##叙述性谬误

叙述性谬误指的是我们无法在不编造理由或者强加一种逻辑关系的情况下观察一系列事实。叙述行为来自一种根深蒂固的简化事物复杂性的生物需要。

叙述谬误及其突出情感事实的特点会扰乱我们对时间概率的预测。看下面一个例子，从概率的角度来考虑，哪种说法更容易发生？

< 1. joy表面上快乐地结婚了。他杀了他的妻子。
< 2. joy表面上快乐地结婚了。他为了得到妻子的遗产而杀了她。

乍看上去第二种情形更有可能发生，而这完全是逻辑的错误。因为第一种情形更宽泛，可以涵盖更多的原因。但是我们更需要一个具体而非抽象的理由来叙述理解这件事情，所以才会认为2更容易发生。类似的决策失误遍布于我们的生活。

情感的力量：一个人的死亡是悲剧，100万人的死亡只是统计学意义上的说法；恐怖主义使人死亡，而最大的杀手仍然是环境，但我们对于人为的毁灭比自然的毁灭更感到痛苦；一个死于骑摩托车的亲戚比大量的统计分析更能影响你对摩托的态度。
