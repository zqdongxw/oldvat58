ubuntu14.04基于L2TP的VPN配置

发表于 2014-10-30 | 分类于 ubuntu |

UBUNTU 自带的VPN配置只有点到点隧道协议（PPTP），那对于vpn的另一种常用协议L2TP，在UBUNTU下又该如何配置呢？近期GAE的不稳定，不得不尝试其他的FQ姿势，于是有了本文的探索和尝试。

方法一： 在UBUNTU networkmanager中安装L2TP插件

sudo apt-add-repository ppa:seriy-pr/network-manager-l2tp  //添加软件源

sudo apt-get update                                                                         //更新软件包缓存

sudo apt-get install network-manager-l2tp                                 // 安装

然后停止并禁用xl2tpd：
sudo service xl2tpd stop sudo update-rc.d xl2tpd disable

reboot，然后在vpn配置里，你会发现L2TP的选项出现了。

方法二：安装l2tp-ipsec-vpn

同样是添加源、更新、安装的步骤：

$> sudo apt-add-repository ppa:werner-jaeger/ppa-werner-vpn
$> sudo apt-get update
$> sudo apt-get install l2tp-ipsec-vpn
 reboot

重启后，你会发现多了一个客户端，点击选择“编辑连接”

[caption id=”” align=”alignnone” width=”438”] ipsec1[/caption]

为你的VPN取一个名字

[caption id=”” align=”alignnone” width=”409”] ipsec2[/caption]

配置IPsec

remote server : VPN的服务器地址, 域名或者IP

use pre-shared key for authentication: VPN的预共享密钥 （下一个选项为导入证书）

[caption id=”” align=”alignnone” width=”466”] ipsec3[/caption]

配置L2TP

[caption id=”” align=”alignnone” width=”464”] ipsec4[/caption]

ppp配置，选择协议，并在此选项卡中填入VPN的用户名和密码。

[caption id=”” align=”alignnone” width=”466”] ipsec5[/caption]

配置结束后，连接VPN。
