php概述

发表于 2012-03-18   |   分类于 PHP   |     |  

Php系列的笔记以这篇概述作开始。我的学习方式主要是视频教程（比如lamp兄弟连的视频），加书籍参考，再加上网上资源的查阅。自学是本人非常推崇的一种态度，整合各种资源会起到事半功倍的效果。

Php是一种可以与各种前台语言（html,javascript等）默契合作的解释型语言。Php的规范标签是<?php ?> ，这是php的官方指定规范特征。其他的简易风格跟变形标签不做介绍，这跟讨论茴香豆的“茴”字有几种写法一样没有意义。在服务器端，php解释器会将<?php和?>的代码进行解释，而不会干扰html跟javascript 。解释完后将整个页面返回给浏览器，在客户端显示出来。

看例子：

Demo.php:

<html>

<title><?php echo”PHP demo”;?></title>

<body>

<script>

alert(new Date());

</script>

<?php echo”<center>”;?>

<h1><?php echo”vat58,成长无限”;?></h1>

<p>

Vat58给你的或许就是一个成长，完善自我的动力！

</p>

<?php

For ($i=0;$i<3;$i++) {

echo”yes,I can. <br>”;

}

?>

</center>

</body>

</html>

上例看出，php真的是“无孔不入”，title标签里， body里，你想要的任何地方都可以。阅读的代码的时候不要被带晕了就好，呵呵。如果你已经迫不及待想学习php的语法，请先行一步学习。上述代码执行的结果如下图:

读者可自行将第一段代码拷贝到一个文件里，命名为*.php,执行并加深理解。

好吧，如果你还没开始php的学习，我再多说几句。你需要一个lamp，wamp,测试环境。比较简单的就是appserv以及其他的集成程序，安装后就可以得到linux/windows+apache+mysql+php测试环境，其他问题请Google自知，或给我留言。
