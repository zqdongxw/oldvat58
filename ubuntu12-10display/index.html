ubuntu12.10安装显卡驱动

发表于 2013-06-05 | 分类于 ubuntu |

ubuntu附加驱动里推荐了几个专有显卡驱动，听网友反映不太靠谱，出现了不少诸如花屏、死机、系统崩溃的情况。本文记录了自己手动安装NVIDIA显卡驱动的过程，希望能帮到你。

[caption id=”attachment_725” align=”aligncenter” width=”300”] ubuntu[/caption]

1. 找到并下载合适的驱动。

用sudo lshw查看硬件中的显卡信息。

description: VGA compatible controller
product: G98 [GeForce G 100]
vendor: NVIDIA Corporation
physical id: 0
bus info: pci@0000:01:00.0
version: a1
width: 64 bits
clock: 33MHz

根据显卡的型号去官网下载匹配的驱动程序，放在主目录下，文件为NVIDIA—xxx.run格式。

2. 卸载系统已有显卡驱动

本来没有进行这一步操作，后来安装英伟达驱动时根据提示需要禁用或卸载已有的驱动。

ERROR: The Nouveau kernel driver is currently in use by your system. This
driver is incompatible with the NVIDIA driver, and must be disabled
before proceeding. Please consult the NVIDIA driver README and your
Linux distribution’s documentation for details on how to correctly
disable the Nouveau kernel driver.

按照安装程序的建议，增加了/etc/modprobe.d/nvidia-installer-disable-nouveau.conf 文件来禁用Nouveau kernel driver，自动的哦。

重启系统后继续进行。

3. 安装编译环境

sudo apt-get install build-essential pkg-config linux-headers-$(uname -r)

4. 启动安装

Ctrl + Alt +F1到第一控制台.输入用户密码login.

sudo stop lightdm 关闭X
sudo sh NVIDIA-xxx.run 运行安装程序



进入安装界面后，首先接受协议，然后编译安装开始，中间很多确认的地方根据实际情况选择， 完成安装，选ok。



sudo start lightdm 启动X


重启系统，是不是清晰了不少？
